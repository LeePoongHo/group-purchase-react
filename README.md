# Sell Log

> Sell Log is a service for group buying.

### Client
- React

### Server
- Node.js
- Koa

### Install yarn
```
$ sudo npm install -g yarn
```

## Getting Started

0. Clone git repository
```
$ git clone https://github.com/LeePoongHo/group-purchase-react
```

1. Install modules
```
$ npm install
```

2. Start React
```
$ npm start
```

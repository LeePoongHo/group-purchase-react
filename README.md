# group-purchase-react
1인가구 공동구매 서비스

## Requirement
- node.js
- yarn

### Install yarn
```
$ sudo npm install -g yarn
```

## Getting Started

0. Clone git repository
```
$ git clone https://github.com/LeePoongHo/group-purchase-react
```

1. Install modules
```
$ npm install
```

2. Install hexo package
```
$ yarn start
```
